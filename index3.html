Bienvenid@s a mi página acerca de mí

Al inicio de mi aprendizaje comencé con los conceptos basicos como lo que es un archivo html y sus usos, mediante este codigo se pueden crear paginas web con tamaño de letra, tipo de letra, algunos colores, también nos permite agrgar imagenes y todo el contenido de nuetra página web, para diseñar estos documentos utilizas visual studio code, a la hora de realizar estos documentos se debe tener mucho cuidado de no equivocarse en las funciones o a la hora de digitar el codigo.

También se hizo uso de otra herramienta que eran los archivos css, que son practicamente aquellos donde se edita nustra pagina los colores, la fuente, imagenes y así la pagina presenta mayor estilo y más estetico. Todo esto nos sirve a la hora de crear alguna página para diferentes funciones como el estudio o algún negocio futuro.
